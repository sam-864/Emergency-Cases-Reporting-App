# Emergency Cases Reporting Mobile App At University

## Project Description
This Android mobile application is designed to help university students and staff
report emergency cases quickly. Users can report incidents such as accidents,
health emergencies, fire outbreaks, and security threats directly through the app.

## Problem Statement
Emergency cases in universities are often reported late due to lack of fast and
reliable reporting systems. This application provides a digital solution to
improve response time and campus safety.

## Objectives
- To enable quick reporting of emergency cases
- To improve emergency response time
- To enhance safety within the university
- To allow users to share emergency details and location

## Main Features
- User registration and login
- Emergency case reporting
- Location sharing
- Notification alerts
- Case status tracking

## Technologies Used
- Platform: Android
- Programming Language: Java / Kotlin
- IDE: Android Studio
- Database: Firebase
- API: Google Maps API

## How to Run the Project
1. Clone the repository
2. Open the project using Android Studio
3. Sync Gradle files
4. Run the app on an emulator or physical device
